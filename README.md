API Lyrics Finder

Um aplicativo web interativo construído com **Python** e **Streamlit** que consome uma API externa para buscar letras de músicas de forma rápida e intuitiva.

![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)
![Streamlit](https://img.shields.io/badge/Streamlit-FF4B4B?style=for-the-badge&logo=Streamlit&logoColor=white)

- Sobre o Projeto
Este projeto foi desenvolvido para praticar a integração de APIs REST em Python e a criação de interfaces de usuário rápidas com o Streamlit. Basta digitar o nome da banda e da música para obter a letra em segundos!
